## Gambaran Umum

**Backend PFS TopUp** adalah REST API untuk aplikasi top-up game dan penjualan akun game. Sistem ini menyediakan:

- Autentikasi berbasis token (Laravel Sanctum)
- Manajemen game dan produk top-up
- Sistem banner dinamis
- Background music management
- Katalog akun game yang dijual
- Role-based access control (Admin & User)

**Base URL Development:** `http://localhost:8000/api`
**Base URL Production:** Disesuaikan dengan deployment Railway/Vercel
